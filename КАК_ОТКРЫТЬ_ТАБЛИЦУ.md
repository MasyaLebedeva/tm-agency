# Как открыть таблицу в Google Sheets

## Способ 1: Импорт CSV файла в Google Sheets

1. Откройте [Google Sheets](https://sheets.google.com)
2. Нажмите на **"Пустой файл"** или **"Blank"**
3. В меню выберите **"Файл"** → **"Импортировать"** (File → Import)
4. Нажмите **"Загрузить"** (Upload)
5. Перетащите файл `ROI_Calculator_Template_GoogleSheets.csv` или выберите его
6. В настройках импорта выберите:
   - **Разделитель:** Запятая
   - **Преобразовать текст в числа, даты и формулы:** ✅ Включено
7. Нажмите **"Импортировать данные"**

## Способ 2: Создать таблицу вручную (проще)

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу
3. Скопируйте структуру ниже:

### Шаг 1: Заголовок
```
A1: T&M Agency - Калькулятор ROI для Telegram-канала
```

### Шаг 2: Услуги (строки 7-15)

| A | B | C | D | E |
|---|---|---|---|---|
| Услуга | Стоимость (₽) | Выбрано (Да/Нет) | Количество | Итого (₽) |
| Telegram ADS | 150000 | Нет | 1 | =IF(C8="Да";B8*D8;0) |
| Посевы | 100000 | Нет | 1 | =IF(C9="Да";B9*D9;0) |
| Креативы | 3000 | Нет | 1 | =IF(C10="Да";B10*D10;0) |
| Оптимизация Telegram-канала | 90000 | Нет | 1 | =IF(C11="Да";B11*D11;0) |
| Разработка Telegram-ботов | 15000 | Нет | 1 | =IF(C12="Да";B12*D12;0) |
| Ведение контента | 20000 | Нет | 1 | =IF(C13="Да";B13*D13;0) |
| Маркировка рекламы | 500 | Нет | 1 | =IF(C14="Да";B14*D14;0) |
| Консультация | 0 | Нет | 1 | =IF(C15="Да";B15*D15;0) |
| **ИТОГО СТОИМОСТЬ УСЛУГ:** | | | | **=SUM(E8:E15)** |

### Шаг 3: Метрики клиента (строки 20-25)

| A | B | C |
|---|---|---|
| Параметр | Значение | Единица измерения |
| Средняя стоимость подписчика (CAC) | | ₽ |
| Средняя стоимость продукта клиента | | ₽ |
| Количество заявок (лидов) | | шт |
| Количество закрытых сделок | | шт |
| Конверсия заявок в сделки (%) | =IF(B24<>0;B25/B24*100;0) | % |

### Шаг 4: Расчет результатов (строки 28-33)

| A | B | C |
|---|---|---|
| Показатель | Результат | |
| Общая стоимость услуг (инвестиции) | =E16 | ₽ |
| Средняя стоимость сделки | =B23 | ₽ |
| Общая выручка от сделок | =B25*B23 | ₽ |
| ROI (Return on Investment) | =IF(E16<>0;(B32-E16)/E16*100;0) | % |
| Прибыль | =B32-E16 | ₽ |

## Важные формулы для копирования:

### В столбец E (Итого по услугам):
```
=IF(C8="Да";B8*D8;0)
```
Скопируйте эту формулу для всех услуг (строки 8-15)

### Итоговая сумма (E16):
```
=SUM(E8:E15)
```

### Конверсия (B26):
```
=IF(B24<>0;B25/B24*100;0)
```

### ROI (B33):
```
=IF(E16<>0;(B32-E16)/E16*100;0)
```

### Прибыль (B34):
```
=B32-E16
```

## Как использовать:

1. В столбце C (Выбрано) укажите **"Да"** для нужных услуг
2. В столбце D (Количество) укажите количество
3. Заполните метрики в разделе "ОСНОВНЫЕ МЕТРИКИ КЛИЕНТА"
4. Все расчеты обновятся автоматически!

## Если формулы не работают:

1. Убедитесь, что вы используете запятую в формулах (не точку с запятой)
2. Проверьте, что ячейки с числами имеют формат "Число", а не "Текст"
3. Если видите формулу как текст, выделите ячейку и нажмите F2, затем Enter
