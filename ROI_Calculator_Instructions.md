# Инструкция по созданию Google-таблицы для расчета ROI

## Шаг 1: Создание таблицы в Google Sheets

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу
3. Назовите её "T&M Agency - Калькулятор ROI"

## Шаг 2: Импорт данных

Или создайте таблицу вручную, используя структуру ниже:

## Структура таблицы

### Раздел 1: Наши услуги и стоимость

| Услуга | Стоимость (₽) | Выбрано (Да/Нет) | Количество | Итого (₽) |
|--------|--------------|------------------|------------|------------|
| Telegram ADS | 150000 | Нет | 1 | `=IF(C2="Да";B2*D2;0)` |
| Посевы | 100000 | Нет | 1 | `=IF(C3="Да";B3*D3;0)` |
| Креативы | 3000 | Нет | 1 | `=IF(C4="Да";B4*D4;0)` |
| Оптимизация Telegram-канала | 90000 | Нет | 1 | `=IF(C5="Да";B5*D5;0)` |
| Разработка Telegram-ботов | 15000 | Нет | 1 | `=IF(C6="Да";B6*D6;0)` |
| Ведение контента | 20000 | Нет | 1 | `=IF(C7="Да";B7*D7;0)` |
| Маркировка рекламы | 500 | Нет | 1 | `=IF(C8="Да";B8*D8;0)` |
| Консультация | 0 | Нет | 1 | `=IF(C9="Да";B9*D9;0)` |
| **ИТОГО СТОИМОСТЬ УСЛУГ:** | | | | **`=SUM(E2:E9)`** |

### Раздел 2: Основные метрики клиента

| Параметр | Значение | Единица измерения |
|----------|----------|-------------------|
| Средняя стоимость подписчика (CAC) | | ₽ |
| Средняя стоимость продукта клиента | | ₽ |
| Количество заявок (лидов) | | шт |
| Количество закрытых сделок | | шт |
| Конверсия заявок в сделки (%) | `=IF(D13<>0;D14/D13*100;0)` | % |

### Раздел 3: Расчет результатов

| Показатель | Формула | Результат |
|------------|---------|-----------|
| Общая стоимость услуг (инвестиции) | `=E10` | ₽ |
| Средняя стоимость сделки | `=D12` | ₽ |
| Общая выручка от сделок | `=D14*D12` | ₽ |
| ROI (Return on Investment) | `=IF(E10<>0;(E18-E10)/E10*100;0)` | % |
| Прибыль | `=E18-E10` | ₽ |

### Раздел 4: Детальный анализ

| Показатель | Значение |
|------------|----------|
| Стоимость привлечения одного подписчика (CAC) | `=D11` |
| Стоимость одной заявки (лида) | `=IF(D13<>0;E10/D13;0)` |
| Стоимость одной закрытой сделки | `=IF(D14<>0;E10/D14;0)` |
| Выручка на одну заявку | `=IF(D13<>0;E18/D13;0)` |
| Выручка на одну закрытую сделку | `=IF(D14<>0;E18/D14;0)` |

### Раздел 5: Выводы и рекомендации

| Показатель | Значение |
|------------|----------|
| Минимальные инвестиции для окупаемости | `=E18` |
| Рекомендуемый бюджет | `=E10*1.2` |
| Ожидаемая прибыль при рекомендуемом бюджете | `=E18-E10*1.2` |

## Шаг 3: Как использовать

1. **Выберите услуги**: В столбце "Выбрано" укажите "Да" для услуг, которые нужны клиенту
2. **Укажите количество**: В столбце "Количество" укажите количество каждой услуги
3. **Заполните метрики клиента**: 
   - Средняя стоимость подписчика (CAC)
   - Средняя стоимость продукта клиента
   - Количество заявок (лидов)
   - Количество закрытых сделок
4. **Просмотрите результаты**: Все расчеты выполняются автоматически

## Формулы для копирования

Если вы создаете таблицу вручную, скопируйте эти формулы:

### Итого по услугам:
```
=IF(C2="Да";B2*D2;0)  // для каждой строки услуги
=SUM(E2:E9)            // для итоговой суммы
```

### Конверсия:
```
=IF(D13<>0;D14/D13*100;0)
```

### ROI:
```
=IF(E10<>0;(E18-E10)/E10*100;0)
```

### Прибыль:
```
=E18-E10
```

## Пример использования

1. Клиент хочет: Telegram ADS, Посевы, Креативы
2. В столбце "Выбрано" ставим "Да" для этих услуг
3. Заполняем метрики:
   - Средняя стоимость подписчика: 500 ₽
   - Средняя стоимость продукта: 5000 ₽
   - Количество заявок: 100
   - Количество закрытых сделок: 20
4. Таблица автоматически рассчитает:
   - Общую стоимость услуг
   - ROI
   - Прибыль
   - Детальные метрики

## Советы

- Чем больше услуг выбрано, тем выше может быть ROI при правильной стратегии
- ROI выше 100% означает, что проект окупился и принес прибыль
- Используйте эту таблицу для презентации клиентам перед началом работы
